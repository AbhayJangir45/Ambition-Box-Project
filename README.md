# Ambition-Box-Project

Job & Salary Insight Platform

A web-based application inspired by AmbitionBox, developed using Python and Flask.
The platform helps users search for jobs, explore company details, analyze salary trends, and gain insights from company reviews.

    • Developed a web-based application using Python and Flask inspired by AmbitionBox.
    
    • Built the platform to help users search for jobs, explore company details, and analyze salary insights.
    
    • Collected and processed company-related data (jobs, salaries, reviews) using Python.
    
    • Performed data cleaning and analysis using Pandas and NumPy . Integrated backend logic with Flask to create a dynamic and interactive       web application.
    
    • Implemented data processing and logic to analyze user preferences and generate relevant travel insights.
    
    • Displayed insights using basic data visualization techniques.


### The application focuses on:

    • Job search and exploration
    
    • Company profile analysis
    
    • Salary trend insights
    
    • Review-based company evaluation
    
    • Data-driven career insights
